Ce dossier contient les fichiers pour le Devoir 3.

Ce projet a été réalisé en MATLAB.
Dans ce projet, la validation d'un code LBM a été étudié.

Les différentes sources d'erreur et leurs incertitudes ont été caractérisées et calculées.

Dossiers ou fichiers :
/doc/CHANGELOG.txt : enregistrement des mises à jour du code
/results : tous les résultats obtenus
/src/Error_bar_graph.m : générer le graphique de l'erreur relative
/src/Generate_sample.m : générer la structure du filtre
/src/launch_simulationLBM.m : exécuter la simulation
/src/launch_simulationLBM_question_B.m : exécuter la simulation pour la partie B
/src/LBM.m : code LBM
/src/LBM_B.m : code LBM pour la partie B
