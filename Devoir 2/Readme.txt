Ce dossier contient les fichiers pour le Devoir 2

Ce projet a été réalisé en Python.
Dans ce projet, le processus de diffusion d'un sel dans un pilier cylindrique poreuse de béton a été étudié.

Le code de différences finies vise à analyser la convergence au moyen d'une comparaison avec une solution
générée dans le logiciel COMSOL et en utilisant la méthode des problèmes proches (MNP).
Ensuite, l'analyse de convergence à l'aide de la méthode des solutions manufacturées (MMS) est réalisée.
